📦 Import ZIP — Dossier complet
Nomme tes fichiers ainsi pour que l'app les classe automatiquement :
CLASSE_MATIERE_TYPE_TITRE.pdf
Ex : Tle-C_Maths_cours_Chapitre5.pdf ou PB_ACD_2022_Travail-Manuel.pdf
✨ Les abréviations sont reconnues (pb, bep, term, tle, 1ere, 2nde...) et tu peux cocher plusieurs classes pour un même fichier après l'import.
📖 Guide — Comment préparer ton ZIP
① Créer le dossier
Crée un dossier sur ton téléphone ou PC.
Mets tous tes fichiers PDF/JPG/PNG dedans.
⚠️ Les sous-dossiers sont aussi lus automatiquement.
② Nommer les fichiers (abréviations acceptées)
Le nom du fichier doit contenir 3-4 informations séparées par _ (underscore). L'app tolère la casse, les accents et de nombreuses abréviations :
${[ ["CLASSE","6e · 5e · 4e · 3e · 2nde/2nd · 1ere/1re · tle/term + série (A/C/D/F) — ex: TleC, term-d, 1ereA. Plusieurs classes/séries possibles: TleACD, PB_C_D"], ["MATIERE","maths/math · fr/francais · ang/anglais · phy/physique · chi/chimie · svt/bio · hg/histoire · philo · info · esp/espagnol…"], ["TYPE","cours · seq · exam · bac · bep/bepc · pb/prob/probatoire · zone · comp"], ["TITRE","ChapitreX · Sujet2024 · Seq1Trim2 · Exercices (libre)"], ].map(([k,v]) => `
${k} ${v}
`).join("")}
③ Exemples de noms valides
${[ ["Tle-C_Maths_cours_Chapitre5-Derivation.pdf","✅"], ["3e_Français_seq_Seq2-Trimestre1.pdf","✅"], ["PB_ACD_2022_Travail-Manuel.pdf","✅ détecte Tle_A + Tle_C + Tle_D d'un coup"], ["term-d_chimie_zone_revisions.pdf","✅ abréviation 'term' reconnue"], ["1ereC-1ereD_Maths_exam_Sujet2024.pdf","✅ 2 classes détectées automatiquement"], ["cours_maths.pdf","⚠️ Classe manquante — tu pourras la cocher manuellement"], ].map(([ex,st]) => `
${st.startsWith('✅')?'✅':st} ${ex} ${st.length>2?`
${st.replace('✅ ','').replace('⚠️ ','')}
`:''}
`).join("")}
④ Compresser en ZIP
Sur Android : Appuie longtemps sur le dossier → Compresser/Archiver → Choisir ZIP
Sur iPhone : Appuie sur le dossier → Partager → Compresser
Sur PC : Clic droit sur le dossier → Envoyer vers → Dossier compressé (ZIP)

✏️ Correction des noms & classes
Scanne tous les fichiers publiés. Corrige le nom si la matière détectée ne correspond pas au type déclaré, et reclasse dans le bon onglet (ex : Histoire classée en Géo → reclassée en Normal).
📅 Classement par année
Attribue automatiquement l'attribut annee aux fichiers dont le titre ou le nom contient une année (ex : 2022, 2023, 2024).
🗑️ Suppression des doublons
Détecte les doublons par titre normalisé identique ET/OU URL de fichier identique. Garde le plus récent, supprime les copies — dans localStorage et dans Turso.
📊 État du contenu publié
Vue d'ensemble de tous les fichiers dans l'app.
📝 Renommer les fichiers (nom propre)
Régénère le champ "nom" de tous les fichiers déjà publiés au format MATIERE_CLASSE_TYPE_ANNEE (ex: MATHS_TleC_EXAM_2026) à partir de leurs métadonnées actuelles (matière, classe, type, année). Le fichier physique sur Cloudinary n'est PAS renommé (l'URL reste inchangée) — seul le nom affiché dans l'app est corrigé.
